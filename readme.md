# SeLoger Scraper

## Description
This repository contains a web scraper for extracting real estate listings from the SeLoger website. The scraper is designed to collect data such as property details, prices, locations, and other relevant information.

## Features
- Scrapes property listings from SeLoger.
- Extracts details such as price, location, and property description.
- Saves the scraped data in a structured format (e.g., CSV, JSON).

## Requirements
- Python 3.8+
- Required libraries are listed in `requirements.txt`.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/seloger_scraper.git
   cd seloger_scraper
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Run the scraper:
   ```bash
   python run.py
   ```
2. The scraped data will be saved in the `data` directory.
